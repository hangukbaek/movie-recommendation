현재까지 작성한 웹페이지 JS 코드(지피티)


1. 장르 버튼 클릭 시 active 클래스 토글 및 콘솔 출력
 기능 요약
장르 버튼을 클릭하면 active 클래스를 동적으로 변경함 (스타일 변경용)

선택된 장르명을 콘솔에 출력함 (API 호출과는 무관)


2. 장르 버튼 클릭 시 API 호출 + 화면 업데이트
기능 요약
장르 버튼 클릭 → 해당 장르 영화 목록 API 호출

받아온 데이터로 카드 목록 UI 갱신

초기 진입 시 '액션' 장르 자동 실행

3. genre.html 페이지에서 URL 파라미터 기반으로 영화 리스트 출력
기능 요약
URL에 담긴 장르 이름(?genre=액션)으로 API 호출

해당 장르의 영화 리스트를 보여줌

영화 클릭 시 상세 페이지로 이동

4. 이주의 HOT 랭킹 슬라이드 기능
 기능 요약
좌우 버튼 클릭 시 .slider-content 영역을 translateX()로 이동

각각 영화 슬라이더(hotMovieIndex), 배우 슬라이더(hotActorIndex) 별도 관리

5.홍길동 님을 위한 추천영화
회원가입을 한 사용자(Ex.윤주상) // 서버 또는 로컬스토리지 등에서 받아올 수 있음
의 정보를 입력 받아 사용자의 이름과 ID를 받아서 "윤주상님을 위한 추천 영화"로 타이틀을 바꾸고, 해당 유저에게 맞는 영화 리스트를 API로 받아 화면에 보여주는 코드

6.검색 기능
검색 버튼 클릭 또는 엔터 입력 시:

입력한 영화 제목을 기준으로 search.html?query=영화제목 페이지로 이동

검색어를 localStorage에 저장

💾 검색 기록 저장 & 표시
최근 검색어는 중복 없이 저장되고, 최대 5개까지만 유지

사용자가 검색어를 입력하면 아래에 검색 기록 목록이 자동 표시됨

❌ 검색 기록 삭제 기능
기록 옆에 있는 X 버튼 클릭 시 해당 검색어가 localStorage에서 삭제되고 즉시 화면에서도 사라짐

📎 검색 기록 클릭 시 동작
저장된 검색 기록을 클릭하면, 해당 검색어로 다시 검색 페이지로 이동
